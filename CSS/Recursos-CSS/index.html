Seletores CSS
SELETOR SIGNIFICADO EXEMPLO
Seletor universal Aplica-se em todos os elementos no documento. *{}
Seleciona todos os elementos da página.
Seletor de texto Seleciona um elemento cujo atributo class tem o valor especificado depois do ponto. h1, h2, h3{}
Seleciona os elementos h1, h2 e h3
Seletor de ID Seleciona um elemento cujo atributo ID tem o valor especificado após o simbolo #. #XXX{}
Selecionao elemento cujo o atributo ID é XXX
Seletor de class Seleciona um elemento cujo atributo class tem o valor especificado depois do ponto. .XXX{}
Seleciona qualquer elemento cujo atributo class tem o valor "XXX"
Seletor de filho Seleciona um elemento que é filho direito de outro. li > a{}
Seleciona quaisquer elementos de a que são filhos de um elemento li (mas não todos os elemento na página)
Seletor de descendente Seleciona um elemento que é descendente de outro elemento especificado (e não apenas filho direito desse elemento). p a{}
Seleciona quaisquer elementos a que residem dentro de um elemento p, mesmo se houver outros elementos aninhados entre eles.
Seletor de irmão adjacente Seleciona um elemento que é o irmão próximo de outro. h1 + p{}
Seleciona o primeiro elemento p depois de qualquer elemento h1 (mas não outros elementos p).
Seletor de irmão geral Seleciona um elemento que é um irmão de outro, embora ele não precise ser o elemento diretamente precedente. h1 ~ p{}
Se houver dois elementos p que fossem irmãos de um elemento h1, essa regra aplica-se ao dois.
