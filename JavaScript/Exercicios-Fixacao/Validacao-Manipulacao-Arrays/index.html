O exercício trata-se de uma validação de elementos adicionados ao input. Caso esteja em branco, será informado que o campo é inválido.

Caso o elemento digitado não conste no Array objetos. Sejá adicionado ao fim.

Para sequenciados em ordem alfabética, basta clicar no botão "Ordernar".

Se for necessário, o input pode ser limpo por intermédio do botão "Limpar".
